# SmartCash 💰

### Seu Dashboard Financeiro Inteligente

O **SmartCash** é uma ferramenta de gestão financeira pessoal desenvolvida para oferecer uma experiência rápida, leve e totalmente privada. A aplicação permite que o usuário monitore suas finanças em tempo real, registrando entradas e saídas de forma simplificada, tudo isso sem a necessidade de cadastros complexos ou conexão com servidores externos.

---

## 🎯 Proposta e Objetivo

O principal objetivo deste projeto é entregar uma **Single Page Application (SPA)** funcional que ajude o usuário a responder à pergunta: *"Para onde está indo meu dinheiro?"*. 

Diferente de planilhas ou aplicativos burocráticos, o SmartCash foca na agilidade:
- Registro instantâneo de transações.
- Cálculo automático de saldo líquido.
- Persistência de dados local, garantindo que as informações fiquem salvas apenas no navegador do usuário.

---

## 📋 Planejamento e Artefatos

A construção do projeto foi precedida por uma etapa de planejamento e prototipagem para definir o escopo e garantir a organização da lógica e da interface.

- **Documento de Requisitos:** Definição completa de funcionalidades e restrições técnicas.
- **Casos de Uso:** Mapeamento das interações do usuário com o sistema.
- **Wireframe:** Estrutura visual planejada com foco em *Mobile First*.

---

## 🛠️ Tecnologias Utilizadas

Para este projeto, optou-se pelo uso de tecnologias nativas para garantir máxima performance e demonstrar domínio sobre os fundamentos do desenvolvimento web:

- **HTML5:** Estrutura semântica para melhor acessibilidade e SEO.
- **CSS3:** Estilização moderna, responsiva e uso de variáveis para guia de estilos.
- **JavaScript Vanilla:** Lógica de programação pura, manipulação dinâmica do DOM e gestão de eventos.
- **Web Storage API (LocalStorage):** Persistência de dados no lado do cliente.

---

## 📂 Estrutura de Arquivos

Seguindo o padrão de separação de responsabilidades, o projeto está organizado de forma simples e direta na raiz:

```text
/
├── index.html    # Estrutura e marcação da página
├── style.css     # Estilização e regras de responsividade
└── script.js     # Lógica, cálculos e manipulação de dados
```
